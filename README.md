# 🌍 Earth Online (地球在線)

Welcome to **Earth Online**, a global idle and social networking experiment. This project is a minimalist, cyberpunk-themed web application where players across the globe become "nodes" in a shared network. The longer you stay online, the more you contribute to the global population's "production" score!

## 🚀 Features

- **Global Live Leaderboard**: See top contributors in real-time, sorted by accumulated idle time and points.
- **Discord OAuth Integration**: Bind your Discord account for quick logins and personalized avatars.
- **Real-Time World Chat**: Communicate with other online nodes globally with zero latency.
- **Friend System**: Send friend requests and build your own network of nodes.
- **Dynamic Milestones**: Global celebrations when the server reaches critical populations.

## 🛠 Tech Stack

- **Frontend**: React + Vite
- **Backend**: Node.js + Express + Socket.io
- **Database**: MongoDB
- **Styling**: Vanilla CSS (Cyberpunk aesthetic)

## 📦 Running Locally

1. Setup your MongoDB and Discord application tokens.
2. Configure `.env` in `/backend` and `.env` in `/client`.
3. `cd backend && npm install && npm start`
4. `cd client && npm install && npm run dev`
